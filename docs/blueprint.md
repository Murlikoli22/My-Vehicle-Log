# **App Name**: MY Vehicle Log

## Core Features:

- User Authentication: Secure user registration and login with email/password and Google Sign-In.
- Vehicle Profile Management: Store vehicle details: registration number, type, brand, model, year, fuel type, and odometer reading.
- Document Storage: Upload and manage vehicle documents (RC Book, Insurance, PUC) with expiry date reminders.
- Maintenance Log: Record service history with dates, odometer readings, service types, mechanic details, costs, and uploaded bills; calculates totals.
- Emergency Mode: Dedicated screen for emergency contacts, medical details, live location sharing, and fullscreen emergency info.
- Smart Reminders: Automated notifications for servicing, insurance, PUC, and other maintenance based on Firestore data.
- Maintenance cost estimation: Tool that leverages user data and historical data on other user vehicles to estimate costs for upcoming service

## Style Guidelines:

- Primary color: Blue (#29ABE2) representing trust and reliability.
- Background color: Light gray (#F2F2F2), providing a clean and neutral backdrop.
- Accent color: Orange (#FF9800) to highlight emergency features and important alerts.
- Body and headline font: 'PT Sans' for a modern, readable style.
- Code font: 'Source Code Pro' for displaying code snippets.
- Simple, clear icons for vehicle types, documents, and maintenance tasks.
- Clean and intuitive layout with easy navigation; bold and prominent design for Emergency Mode.
